# Personal Portfolio - Hemawarshini Mahendran

A modern, responsive personal portfolio website built with **React + Vite** using **plain CSS**.

## Features

- Responsive portfolio with modern UI
- Sections: Navbar, Home, About, Education, Projects, Skills, Contact, Footer
- Dark/Light mode toggle with localStorage persistence
- Smooth scrolling navigation
- Animated hero section and hover effects
- Project cards and skill progress bars
- Download Resume button

## Tech Stack

- React
- Vite
- Plain CSS (Flexbox + Grid)

## Run Locally

1. Install Node.js (LTS) from https://nodejs.org/
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start development server:

   ```bash
   npm run dev
   ```

4. Build for production:

   ```bash
   npm run build
   ```

5. Preview production build:

   ```bash
   npm run preview
   ```
